Introduction

The Compliance Check system is a powerful tool that helps users ensure compliance with local and international sanctions regulations and also enable users to see if a customer is under special list. Before opening an account for a new customer, users can use the system to search multiple lists and verify that the customer is not subject to any restrictions or if he/she need special attention.

One of the key features of the Compliance Check system is its ability to perform fuzzy searches. This means that even if a user enters a name with a slightly different spelling, the system will still be able to find potential matches and display them along with a score indicating the level of similarity. This makes it easy for users to find the information they need, even if they are not sure of the exact spelling of a name.

Overall, the Compliance Check system is an essential tool for ensuring compliance and reducing risk when opening new accounts.

Features

The Compliance Check system offers a range of powerful features to help users ensure compliance with local and international sanctions regulations and other lists. Some of the key features of the system include:

  • Multiple Lists: The system can search multiple local and international lists, including UN individuals and entities, EU sanction list, UK sanction list, OFAC list, and local lists such as NBE delinquent list, NBE block list, FIS block list, media adversary list, HPR, and politically exposed people.
  • Fuzzyd Search: The system can perform fuzzy searches to find potential matches even if the user enters a name with a slightly different spelling. The results are displayed along with a score indicating the level of similarity.
  • Search by Name: Users can search for individuals or entities by name to quickly and easily verify their compliance status.
  • Score by Percentage: The system displays a score by percentage to indicate the level of similarity between the search query and the results. This makes it easy for users to assess the relevance of the results and make informed decisions.

Overall, the Compliance Check system offers a range of powerful features to help users ensure compliance and reduce risk when opening new accounts.
