Get Started

When you log in you see dashboard. if you are allowed to use compliance search module you should be able to see compliance check in side bar as you see in the picture. inside it click on search to start using the system

when you start using the system for the first time the system takes some time to load all the data. wait patiently and you can start searching while it loads but make sure that you don't close the tab while it loads

To search for a user you type the name in search bar.
